h1. Location Profiler

Ruby on Rails RESTful web app with RESTful api.  Backend to Location Profiler iOS app.  

Location Profiler app will allow users to answer questions and based on their answers recommend locations where people they might get along with hang out frequently.

Backend creates api that is consumed by front end.

Uses:

* devise gem - authentication
* omniauth with devise - facebook authentication
* Core Data
* Rest Kit
* Map Kit
* Custom Annotations
* UITableView
* UITabBarController
* Custom Table Cells
* Blocks
* Delegates
